# .github

This repository contains organization-wide files and configuration for the Fitness-Wiz organization.

## Contents

- `profile/` - Organization profile README displayed on the organization's GitHub page
- Community health files (CODE_OF_CONDUCT, CONTRIBUTING, SECURITY) - Shared across all repositories

## About Fitness-Wiz

**Fitness-Wiz** is an independent fitness product focused on workout tracking and routine management.

This organization contains the codebase for the mobile application, backend services, and internal tools used to build and operate the product.

The project is actively developed with an emphasis on clean architecture, maintainability, and real-world scalability.

Some repositories are private while development is ongoing.
