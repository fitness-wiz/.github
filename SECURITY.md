# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within any Fitness-Wiz repository, please send an email to security@fitness-wiz.com (or create a private security advisory on GitHub).

Please do not publicly disclose the issue until it has been addressed by our team.

## Supported Versions

We release patches for security vulnerabilities. Please ensure you are using the latest version of our applications.

## Security Response

We will acknowledge your report within 48 hours and provide a detailed response indicating next steps within 5 business days.

We will keep you informed of the progress towards a fix and public disclosure, and may ask for additional information or guidance.

Thank you for helping keep Fitness-Wiz and our users safe!
